件_l注意c：
1、文件信息声明位于头文件和定义文件的开头（参见示例1-1），
主要内容有：
（1） 版权信息；
（2） 当前版本号，作者/修改者，完成日期；
（3） 主要函数描述。
示例1-1
/**
 * Copyright(c)2013,Wozlla,www.wozlla.com
 * Version: 1.0
 * Author: Peter Zhang
 * Date: 2013-02-17
 * Description: main functions
 */
2、变量的命名规则要求采用“小驼峰式命名法”
3、代a行
3.1、一行代码只做一件事情，
3.2、if、for、while、do等语句自占一行，执行语句不得紧跟其后，不论执行语句有多少都要加{}，这样可以防止书写失误；
3.3、所有函数内的变量都在函数开始处定义；尽可能在定义变量的同时初始化该变量(就近原则)
3.4、代码的的对齐采用TAB键而不采用空格键对齐
3.5、长行拆分
3.5.1、代码行最大长度宜控制在70至80个字符以内；
3.5.2、长表达式要在低优先级操作符处拆分成新行，操作符放在新行之首（以便突出操作符），拆分出的新行要进行适当的缩进，使排版整齐，语句可读。
